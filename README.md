# Ejercicios de Spring MVC

Este repositorio contiene ejercicios prácticos para acompañar un curso de Spring MVC.  
El objetivo es practicar paso a paso los conceptos principales del desarrollo de aplicaciones web con Spring.

## Objetivo del repositorio

A lo largo de estos ejercicios aprenderás a:

- Crear controladores con Spring MVC.
- Gestionar rutas y peticiones HTTP.
- Enviar datos desde el controlador a la vista.
- Trabajar con formularios.
- Validar datos introducidos por el usuario.
- Organizar una aplicación web siguiendo una estructura clara.
- Conectar Spring MVC con servicios y repositorios.
- Preparar pequeños proyectos web completos.

## Requisitos previos

Antes de empezar, es recomendable tener conocimientos básicos de:

- Java
- Programación orientada a objetos
- HTML básico
- Maven
- Conceptos básicos de aplicaciones web

También necesitarás tener instalado:

- Java JDK
- IntelliJ IDEA
- Maven
